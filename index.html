애플리케이션 설정

AI 검색 및 분석에 사용될 구글 Gemini API 정보를 설정합니다.

API Key 발급받기

입력하신 API 키는 브라우저 내부 스토리지(localStorage)에만 안전하게 저장되며 외부 서버로 무단 전송되지 않습니다.

eBird 토큰 발급받기

eBird 토큰을 설정하시면 새를 식별했을 때 해당 새의 뉴질랜드 내 최근 실제 관찰 보고 위치 목록(최근 30일)을 조회하여 지명과 날짜를 띄워줍니다.
